## 项目背景

希望用治愈系 UI 包装开发者数据，让作品集不再冷冰冰。

## 技术方案

- React 19 + Vite 构建
- animal-island-ui 组件库
- MDX 管理内容与案例说明

## 成果

首屏加载 < 2s，移动端适配完整，支持暗色季节 overlay。

```tsx
import { Card, Title } from 'animal-island-ui';

<Card color="app-blue">
  <Title>Hello Island</Title>
</Card>
```
