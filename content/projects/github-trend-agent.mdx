## 概述

对话式 Agent：用户提问后自动采集 GitHub 近 7 天活跃热门项目，按综合数据得分排序，并由 DeepSeek V4 生成解读。

## 亮点

- GitHub Search API 实时数据，stars / forks 综合得分排榜
- 默认 DeepSeek V4 Pro 模型，可切换 Flash
- 动森风格对话界面 + 数据榜双栏布局
